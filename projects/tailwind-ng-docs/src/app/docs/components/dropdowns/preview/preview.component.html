Accessibility features: Inspired by the combobox pattern from the ARIA Authoring Practices Guide (APG)
  • When the dropdown is opened, press Espace/Esc key to close it.
  • When the dropdown is closed, press Espace/Esc key to reset selection and clear text field.
  • When the dropdown is opened, press Up/Down arrow keys to navigate through the list.
  • When the dropdown is opened, press Enter/Space key to select the focused item.
